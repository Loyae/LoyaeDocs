 
Frequently Asked Questions
==========================

This section addresses common questions and issues.

How do I report a bug?
----------------------

If you find a bug in MyProject, please report it on our GitHub issue tracker:

- `GitHub Issues <https://github.com/username/myproject/issues>`_

Can I contribute to MyProject?
------------------------------

Yes, contributions are welcome! Please see our `contributing guidelines <https://github.com/username/myproject/blob/main/CONTRIBUTING.md>`_ for more information.

What versions of Python are supported?
--------------------------------------

MyProject supports Python 3.6 and later. We recommend using the latest stable version of Python for the best performance and features.

Where can I find more help?
---------------------------

If you need more help, you can:

- Check the `official documentation <https://myproject.readthedocs.io>`_
- Ask a question on our `community forum <https://community.myproject.com>`_
